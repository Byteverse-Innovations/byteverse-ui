# Real estate site delivery checklist — WordPress

Parallel checklist for Kadence/Astra + IDX plugin. Assumes **IDX via an approved vendor** (embed/widget/deep links), not a custom MLS backend.

## Discovery & access

- [ ] Confirm IDX provider (**one**: iHomefinder / Showcase IDX / IDX Broker) + subscription tier
- [ ] Confirm MLS allows required displays (featured, map search, saved search/alerts, REO filter—**if promised**)
- [ ] Staging URL + production domain/DNS plan; SSL

## Core build

- [ ] Install WP, harden basics (updates strategy, backups, admin users)
- [ ] Theme + child theme; global styles (fonts, colors, logo)
- [ ] Mobile-first templates for Home, interior pages, landing pages
- [ ] Homepage: hero slideshow, IDX featured block, quick search widget, testimonials carousel, CTAs
- [ ] Full IDX search page (map/list) per vendor setup
- [ ] Regional strategy: **5 community pages** + internal links; each with IDX embed/search deep link
- [ ] **Interactive Florida map**: 5 regions → verified filtered IDX URLs
- [ ] Buyers hub + lead capture form
- [ ] Sellers hub + home value/CMA landing + forms
- [ ] “Market data” page: agreed format + data source
- [ ] Specialty pages: pre-foreclosure, short sale, foreclosure/REO search (**feasibility verified**)
- [ ] About/bio, testimonials page, contact (click-to-call, email, social, form)

## Plugins & integrations (keep the set minimal)

- [ ] IDX plugin configured + pages wired + disclaimers/footer compliance text as required
- [ ] Forms + spam protection + notifications/CRM routing
- [ ] Mortgage calculator embedded (**no unwanted redirects**)
- [ ] Rank Math: titles/metas, sitemap, breadcrumbs (if used), OG basics, critical schemas (LocalBusiness/Person if applicable)
- [ ] Analytics (GA4) + conversion events (form submits)

## Performance & QA

- [ ] Image optimization + lazy loading; WebP where appropriate
- [ ] Core Web Vitals spot-check (mobile)
- [ ] Cross-browser + **iPhone/Android** pass on key flows (search, forms, maps/embeds)

## Launch

- [ ] Production migration checklist (URLs, permalinks, embed domains allowlists if needed)
- [ ] Search Console property + sitemap submit
- [ ] Google Business Profile: claim/create guidance + link website (**client completes verification steps**)
- [ ] Redirect plan if replacing an old site
- [ ] Client handoff: editor training, plugin update expectations, support window defined

## Launch Package (client-facing sub-lines)

Use under the main **Launch Package** line item on quotes; maps to the checklist above without overwhelming scope.

**Launch Package** — WordPress real estate site (Kadence or Astra + one IDX provider)

1. **Setup & IDX** — WordPress install and baseline hardening; confirm IDX vendor, tier, and what’s allowed (search, featured, alerts, REO filter if included); staging, domain/DNS, SSL.
2. **Design system** — Child theme, brand styles (fonts/colors/logo), responsive layouts for key templates.
3. **Homepage** — Hero slideshow (client photography), featured listings from IDX, quick search, testimonial carousel, clear CTAs.
4. **Search & regions** — Full IDX search (map/list per vendor); five community pages with local content and IDX links; interactive Florida map with five regions to those searches.
5. **Buyer & seller tools** — Buyers page + lead capture and IDX flows as supported; sellers page + home value/CMA landing + forms; embedded mortgage calculator (no off-site redirect for the tool).
6. **Specialty & market** — Market data page (format agreed in kickoff); pre-foreclosure, short sale, and REO-focused search **subject to MLS/IDX rules**.
7. **Trust & contact** — About/bio, testimonials (page + homepage), contact with call, email, social, and form.
8. **SEO, QA & launch** — Rank Math basics, forms + spam protection, GA4 + form events; image/performance pass and iPhone/Android check on search, forms, embeds; go-live, Search Console + sitemap, GBP setup guidance (client finishes verification), handoff and short training.

*IDX subscription, domain purchase, and verified Google Business Profile are client responsibilities; we configure and guide.*
